
#📊 E-Commerce Sales Dashboard | Power BI

📌 Project Overview

This project presents an interactive E-Commerce Sales Dashboard developed using Microsoft Power BI. The dashboard transforms raw sales data into meaningful business insights through data cleaning, modeling, KPI calculations, and interactive visualizations. The primary objective is to help business stakeholders monitor sales performance, identify trends, and make informed, data-driven decisions.

The dataset was first cleaned to remove duplicates, handle missing values, standardize formats, and improve overall data quality. After the cleaning process, the refined dataset was imported into Power BI for visualization and analysis.

🎯 Key Performance Indicators (KPIs)

The dashboard highlights the following business metrics:

- Total Revenue: 5.92M
- Total Orders: 46
- Average Order Value: 128.75K

These KPIs provide a quick overview of the organization's sales performance.

📈 Dashboard Features

The dashboard includes multiple interactive visualizations, such as:

- Revenue by Product (Bar Chart)
- Revenue by Region (Pie Chart)
- Product Revenue Share (Donut Chart)
- Units Sold by Product (Column Chart)
- Units Sold by Salesperson (Bar Chart)

Interactive slicers for Salesperson, Product, and Region enable users to dynamically filter the dashboard and explore insights from different perspectives.

🛠️ Tools & Technologies

- Microsoft Excel (Data Cleaning)
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

📊 Business Insights

The dashboard helps users:
